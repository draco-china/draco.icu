    学习的最好途径就是看书

    开篇想给大家两点关于读书的建议：

    经典书籍需要不断被重读——每一次重读都会有新的体会；

    书籍并非读的越多越好——大多数书籍只是经典书籍中的概念延伸（有时甚至是照搬）

    JAVASCRIPT

    1. 《JavaScript 高级程序设计》
    2. 《JavaScript 忍者秘籍》
    3. 《你不知道的 JavaScript》 上 中 下
    4. 《JavaScript 设计模式》
    5. 《JavaScript 设计模式与开发实践》
    6. 《ECMAScript 6 入门》

    TypeScript

    1. 《深入理解 TypeScript》
    2. 《重学 TS》

    CSS

    1. 《CSS 揭秘》
    2. 《CSS 世界》

    Node

    1. 《深入浅出 Node.js》
    2. 《Node.js：来一打 C++ 扩展》
